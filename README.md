# Weather App

A clean weather app that shows current weather conditions for any city.

## Features

- City search with keyboard support (Enter key)
- Displays temperature, humidity, wind speed, and feels-like
- Recent searches saved to localStorage
- Error handling for unknown cities
- Glassmorphism UI design

## Tech Stack

- HTML5
- CSS3 (Glassmorphism, backdrop-filter)
- Vanilla JavaScript
- localStorage for persisting recent searches

## Getting Started

Open `index.html` in your browser. Search for cities like Mumbai, Delhi, London, New York, Tokyo, Paris, Sydney, Bangalore.

> Note: Uses mock weather data. To use live data, replace with your OpenWeatherMap API key.
